Pobierz entry ID z formularza Google

Żeby formularz na stronie zapisywał odpowiedzi do Twojego formularza Google, w pliku script.js muszą być wpisane prawdziwe entry ID pól. Pobierz je tak:

  1. Otwórz swój formularz w widoku wypełniania (nie edycji):
    Otwórz formularz kontaktowy
  2. Wciśnij F12 (DevTools) → zakładka Console.
  3. Wklej poniższy skrypt i naciśnij Enter.
  4. W konsoli zobaczysz listę entry.XXXXXX w kolejności pól (Imię, Nazwisko, Email, Telefon, Wiadomość). Skopiuj je do script.js w zmiennej GOOGLE_FORM_ENTRIES.

[...document.querySelectorAll('[data-params]')]
  .map(el => {
    const m = el.dataset.params && el.dataset.params.match(/\[\[(\d+)/);
    return m ? `entry.${m[1]}` : null;
  })
  .filter(Boolean)
  .forEach((id, i) => console.log(id));
